# Leather Manufacturing & Export Analytics

## Overview
An end-to-end Data Analytics project developed using synthetic data representing a leather manufacturing and export company in Tamil Nadu.

## Tools
- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- SQL
- Power BI

## Features
- Executive Dashboard
- Production Analytics
- Quality Analysis
- Inventory Dashboard
- Shipment Tracking
- Financial KPIs
- Employee Performance

## Dataset
Synthetic dataset with approximately 30,000 records across multiple business domains.

## How to Run

pip install -r requirements.txt

streamlit run app.py
